
# Maruthi HR Solution | Payroll Portal

A secure, cloud-ready payroll management system designed for HR Consultants to manage multiple Principal Employers and generate individual or bulk payslips.

## 🚀 Features
- **Multi-Client Support**: Manage separate payroll databases for multiple companies (Principal Employers).
- **Automated Payslip Generation**: Import CSV/Excel ledgers and instantly generate professional PDF payslips.
- **Smart Mapping**: Intuitively maps your CSV columns to the portal fields.
- **Financial Insights**: Real-time analytics on CTC payouts, PF funds, ESI contributions, and net salary trends.
- **Secure Access**: AES-256 cloud-simulated encryption for data and multi-attempt login protection.
- **Statutory Compliance**: Automated calculation of PF, ESI, PT, and LWF deductions.

## 🛠 Setup & Usage
1. **Login**: Use default admin credentials (`admin` / `admin123`).
2. **Setup Clients**: Add your Principal Employers using the (+) button in the sidebar.
3. **Import Data**: Select a client, click "Import Monthly Ledger", and map your CSV columns.
4. **Distribute**: Go to the "Payslips" tab to search, view, and download individual or bulk PDFs.

## ☁️ Deployment
This project is optimized for deployment on **Vercel**, **Netlify**, or **GitHub Pages**.
- Simply connect this repository to your hosting provider.
- The app uses ES6 modules and requires no complex build step for basic hosting.
